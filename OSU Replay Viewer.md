# OSU Replay Viewer

Goal: Be able to view and edit osu replay files, then export them as .osr files

- Input .osr file
- Process it
- Allow the client to **view** the process replay
- Allow the client to **edit** the replay
- Export
	- As .osr or maybe as a video
