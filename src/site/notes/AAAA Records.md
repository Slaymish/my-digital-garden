# AAAA Records

- Resolve [[IPv6\|IPv6]] address records
- Map a hostname to a IPv6 address in the [[DNS\|DNS]]

- Similar to [[A Records\|A Records]]

```
www.example.com. AAAA 2001:db8:85a3:0:0:8a2e:0370:7334
```
