# Enums in Java

```java
public enum DayOfWeek {
	MONDAY, 
	TUESDAY, 
	WEDNESDAY, 
	THURSDAY, 
	FRIDAY, 
	SATURDAY, 
	SUNDAY 
}
```

- These constants can be accessed using dot notation, such as `DayOfWeek.MONDAY` or `DayOfWeek.SUNDAY`
