# Kruskal's Algorithm

- Results in a minimum [[Spanning Trees\|Spanning Trees]]

## Steps

- Look for **smallest edge** in edge list
- If it doesn't cause a [[Cycles\|cycle]], add edge
- Keep going until all nodes in same tree

## Time Complexity

$$O(Elog(E))$$
